The purpose is to create an Open Educational Resource (OER) focused on "Memory, the ability to remember." Developed under the guidance of a field expert professor, it leverages investigative AI to build the knowledge base and online application. It features descriptive modules with standardized interactive tests to measure various memory types, enabling collective analysis and comparison of participant groups based on sociodemographic data.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

To connect a domain, navigate to Project > Settings > Domains and click Connect Domain.
